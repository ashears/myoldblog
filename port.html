LinearAlgebraTutor

Interactive web application designed to teach linear algebra in an accessible manner. Each topic module has a section where the user can reinforce their learning by answering questions. Project repo available at LinearAlgebraTutor Repo

SecureCal

A Calendar application designed from the ground up with data security in mind. With SecureCal users can feel safe knowing that their data never gets sent over the internet, and is encrypted using modern cryptographic techniques before being stored on the user's computer.
Created as a team project for CIS 422: Software Methodology at the University of Oregon.

DuckChat

DuckChat allows users to send datagram messages to other users over a command-line interface. Features:

  • Each user can choose a username
  • Users choose to join or create different channels
  • Duckchat servers can be configured with customized network topography

StudyAid

StudyAid is a desktop application that allows users to build a custom set of questions to test, and then gives them the ability to test themselves using these flashcards. StudyAid was developed using the Unity Engine, and is coded in C#.
